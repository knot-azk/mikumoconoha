  このプラグインをインストールすれば、そのアカウントの本来の名前が表示されるようになります。

* インストール方法
  以下のコマンドを実行してください。

#+BEGIN_SRC sh
  $ mkdir -p ~/.mikutter/plugin/; git clone https://github.com/toshia/tomorinao.git ~/.mikutter/plugin/tomorinao/
#+END_SRC

* 使い方
  正常にインストールできていれば快適な友利奈緒ライフを送れると思います。
